projects

corgi
Go GitHub GitBook

Corgi is an HTML template engine for Go. Thanks to its pug-like syntax, templates are highly readable, while also expanding upon pug's capabilities.

It uses code generation, making template execution fast, while also benefiting from compiler checks for all Go expressions used inside it, making errors less likely.

Currently, corgi is used successfully at X-LAN for many web apps.

dismock
Go GitHub

Dismock is a helper for mocking requests to the Discord API.

Most of dismock's source code is generated by parsing the Go source of of arikawa's API methods and generating mocks from them. Only more complicated API requests are handwritten.

This keeps maintenance minimal and makes keeping up-to-date incredibly easy.

adam
Go GitHub GitBook

Adam was an advanced Discord bot framework featuring localization, command grouping, and extensive set of utilities.

Unfortunately, shortly after completion it became obsolete with Discord's release of slash commands and the subsequent restrictions around the 'Read Messages' Intent.
